# [每周一书] 汪曾祺《人间有味》

美食是热爱生活的方式。集合人生之流、岁月场景、与世间万物的喜悲之心于一茶一盏，一饭一蔬中

该书汪曾祺写美食与爱的经典散文集，在他的文字里，生活是很有意思，有滋有味的。在他的笔下，简单的一茶一饭，一人一事，都洋溢着情趣。汪曾祺先生说自己是很爱逛菜市场的。看看生鸡活鸭、鲜鱼水菜、碧绿的黄瓜、彤红的辣椒，热热闹闹、挨挨挤挤，让人感到一种生之乐趣。
